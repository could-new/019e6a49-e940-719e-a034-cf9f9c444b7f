# Translator App

A clean, responsive translation application built with Flutter.

## Features
* **Responsive Layout:** Automatically adapts to mobile, tablet, and desktop screen sizes.
* **Language Selection:** Choose source and target languages from a pre-defined list.
* **Swap Functionality:** Easily swap source and target languages.
* **Mock Translation Engine:** Simulates translation delays and outputs placeholder text (ready to be connected to a real translation API).

## Tech Stack
* Framework: Flutter
* Language: Dart
* Supported Platforms: iOS, Android, Web, macOS, Windows, Linux

## Setup
1. Ensure you have Flutter installed (version 3.7.2 or higher).
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the application on your preferred device or emulator.
